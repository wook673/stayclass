# 영상메이커 사용법

주제 하나로 "대본 → 음성 → 자막 → 스톡영상 → 연출 → 완성 MP4"를 만드는 프로그램.
추가 비용 0원 (Claude Max 구독 + 무료 도구만 사용).

## 실행

`video-maker` 폴더의 **start.bat 더블클릭** → 브라우저가 자동으로 열림 (http://127.0.0.1:5000)

---

## 기본 흐름 (가장 많이 쓰는 순서)

### A. 주제만 있을 때 (버튼 한 번으로 대본 완성)
1. UI 왼쪽 **[+ 새 프로젝트]** → 이름 입력
2. **0번 카드**: 주제 입력 → 분량 선택 → **[🤖 대본 자동 생성]** 클릭
   (Claude Code CLI를 내부에서 호출 — Max 구독 포함, 20초~1분, 추가비용 0원)
3. 완성되면 1번 칸에 대본이 자동으로 채워짐 → 확인/수정
4. **2번 카드**: TTS 목소리 선택(남/녀) 클릭 → 음성 생성
5. **3번 카드**: [🎬 영상 만들기] → 1~3분 후 완성
6. **4번 카드**: 미리보기 / [⬇ 다운로드]

* 장면별 고급 연출(direction.json)까지 원하면: 0번 카드의 "수동으로 요청문 복사" →
  Claude Code 채팅에 붙여넣기 (대본+연출을 함께 작성해 줌)

### B. 대본을 직접 쓸 때
1. 새 프로젝트 → **1번 카드**에 대본 입력 (자동저장, 글자수·예상길이 표시)
2. 이후 A의 5~7과 동일

### C. 내 목소리로 만들 때
- 대본을 읽어 녹음(휴대폰 등) → **2번 카드 [🎙️ 녹음 올리기]**로 업로드
- 나머지 동일. (파일 이름은 아무거나 OK)
- 참고: TTS는 자막 오타 0%, 녹음은 받아쓰기라 드물게 오타 가능

## 연출 설정 (2.5번 카드)

| 항목 | 설명 |
|------|------|
| **스타일** | docu(다큐·느린줌·암전) / vlog(발랄·슬라이드) / info(정보·깔끔) / brand(브랜드·고급) — 장면 전환·자막 톤이 통째로 바뀜 |
| **강조 단어** | 쉼표로 입력 (예: `92%, 420만 원`) → 그 단어가 나오는 자막에서 골드색 강조 |
| **배경음악** | 드롭다운에서 라이브러리 곡 선택(또는 직접 업로드) → 자동 믹스. 내레이션 나올 때 음악이 자동으로 작아짐(덕킹) |

### BGM 라이브러리
- 음악 파일을 **`video-maker\music\`** 폴더에 넣으면 → 모든 프로젝트의 드롭다운에 나타남 (한 번만 넣으면 계속 재사용)
- 무료 음원 추천: **유튜브 오디오 라이브러리**(유튜브 스튜디오 안, 수익화 안전), **Pixabay Music**(출처표기 불필요)
- 우선순위: 프로젝트에 직접 올린 곡 > 라이브러리 선택 곡

## 고급 연출 (direction.json — Claude에게 부탁)

장면별 세밀 연출은 프로젝트 폴더의 `direction.json`으로 제어. 직접 안 짜도 되고,
채팅에서 이렇게 부탁하면 Claude가 만들어 줌:

> "프로젝트 OO에 연출 짜줘 — 숫자 나올 때 카운트업, 대표 소개 장면에 로어써드"

가능한 연출:
- 장면별 **전환효과**: fade, fadeblack(암전), slideleft, zoomin, circleopen 등
- **자막 애니메이션**: fadeup(떠오름), pop(통통), 중앙 타이틀(position: center)
- **모션그래픽**(overlays): countup(숫자 카운트업), lowerthird(이름·직함 바), titlecard(오프닝 타이틀)
- 사진 소재는 **켄번스**(줌·팬)로 자동으로 살아 움직임

## 채팅(Claude Code)으로 할 수 있는 것

| 하고 싶은 것 | 채팅에 이렇게 |
|---|---|
| 주제로 대본 받기 | UI 0번 카드의 복사 버튼 사용 (권장) |
| 대본 수정 | "OO 프로젝트 대본 더 짧게/자극적으로 고쳐줘" |
| 연출 만들기/수정 | "OO 연출 짜줘 / 전환 더 빠르게 바꿔줘" |
| 문제 해결 | 오류 메시지 그대로 붙여넣기 |
| 이어서 작업 | "영상메이커 이어서 하자" |

## 명령어(터미널파)

```
python make_video.py --new 이름       새 프로젝트
python make_video.py 이름             영상 제작
python make_video.py 이름 --from 4    4단계(클립)부터 재시작
python generate_tts.py 이름 InJoon    TTS만 생성 (SunHi=여성)
python estimate.py 이름               글자수·예상 길이
```

## 폴더 구조 / 산출물

```
projects/<이름>/
  script.txt       대본
  settings.json    스타일·강조어·주제 (UI에서 저장됨)
  direction.json   장면별 연출 (선택, Claude 작성)
  voice.mp3        TTS 음성 (+voice.words.json 타이밍)
  recording.*      직접 녹음 (올린 경우)
  bgm.*            배경음악 (올린 경우)
  assets/ work/    소재·중간파일 (지워도 됨, 재생성됨)
  final.mp4        ★ 완성 영상
```

## 자주 묻는 문제

- **영상에 엉뚱한 스톡이 나옴** → 키워드 자동매칭 한계. 채팅에서 "OO 장면 소재 바꿔줘" 요청
- **자막 오타** → TTS 사용 시 0%. 본인 녹음은 또렷하게 읽으면 개선
- **음성 없음 오류** → 2번 카드에서 TTS 생성 또는 녹음 업로드 먼저
- **카운트업이 안 나옴** → direction.json에 overlays가 있어야 함 (Claude에게 부탁)
- **처음부터 다시** → 프로젝트 폴더에서 work/ 삭제 후 다시 만들기
